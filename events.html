Event Modifications

Trial Events

Trial events are events that don't count towards team score. However, individuals will still be receiving medals for most trial events (check event modifications). We will be hosting a variety of fun trial events. The event supervisors worked hard to make these tests fun and engaging, so we encourage all teams to do these events.
